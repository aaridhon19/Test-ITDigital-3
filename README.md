# Test IT Digital Level 3

## Clone Repository

Clone Repository to your device

```bash
    git clone https://github.com/aaridhon19/Test-ITDigital-3.git
    cd Test-ITDigital-3
```
    
## Installation

Install with npm

```bash
    npm install
```

## Run

```bash
    node index
```

```bash
    npm start
```

Nodemon

```bash
    npm run dev
```
